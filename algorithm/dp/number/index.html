数位 DP

2025-11-30

Intro

数位 DP 类型的题目一般有一种特征：
某一个区间内满足某种特定条件的数的个数。

tricks

  • 将二维区间 [L,R][L,R][L,R] 转换为一维。
    可以设 fif_ifi​ 表示从 000 到 iii 之间的所求数的个数，则问题等价于 fR−fL−1f_R-f_{L-1}fR​−fL−1​。

  • 从树的角度考虑。
    将一个 nnn 位整数 NNN 看作 an−1an−2…a0‾\overline{a_{n-1}a_{n-2}\dots a_0}an−1​an−2​…a0​​。
    这时候可以挨个考虑，例如 an−1a_{n-1}an−1​ 这一项可以分为 [0,an−1−1][0,a_{n-1}-1][0,an−1​−1] 和 an−1a_{n-1}an−1​（对于新数来说），以此类推，形成一种树状结构，而左侧分支可以用组合数直接求出，右侧分支只剩余 a0a_0a0​，便可以直接得出答案。

Examples

Acwing 1081. 度的数量

题目传送门
将题目转换一下，会变成求区间 [L,R][L,R][L,R] 在 BBB 进制下有 KKK 个 111 的方案数。
我们要求从 000 到 (x)B(x)_B(x)B​ 的方案数，我们从头到尾考虑，设 resresres 为当前答案，lastlastlast 表示填过的 111 的数量。
若 xi>0x_i > 0xi​>0，即存在左分支，当前位填 000 时方案数为 (iK−last)\binom{i}{K-last}(K−lasti​)，否则只能填 111，方案数为 (iK−last−1)\binom{i}{K-last-1}(K−last−1i​) 且不存在右分支（当前位不能填 xix_ixi​）。
若最后存在右分支，即 i=ni=ni=n 且 last=Klast=Klast=K，则有额外的一种方案。

Acwing 1082. 数字游戏

题目传送门
依旧要求 000 到 nnn 的方案数，但左侧分支的意义改变了为求最高位为 jjj，位数为 iii 的不降数个数。
考虑使用递推处理左分支方案数，设 fi,jf_{i,j}fi,j​ 表示最高位为 jjj，位数为 iii 的不降数个数。
根据第二位来划分，发现 fi,j=∑k=j9fi−1,kf_{i,j}=\sum\limits _{k=j}^9 f_{i-1,k}fi,j​=k=j∑9​fi−1,k​，直接预处理即可。
至于最右侧的分支，则是直接判断 NNN 是否为不降数。

Acwing 1083. Windy数

题目传送门 同上，由于不能有前导 000，所以在 an−1a_{n-1}an−1​ 上只能填 111 到 xn−1x_{n-1}xn−1​，其他分支可以填 000。
令 fi,jf_{i,j}fi,j​ 表示最高位为 jjj，共有 iii 位的 Windy 数个数，根据第二位划分可以得到： fi,j=∑k=09fi−1,k(∣j−k∣≥2)f_{i,j}=\sum\limits _{k=0}^9 f_{i-1,k}(\left|j-k\right|\geq 2)fi,j​=k=0∑9​fi−1,k​(∣j−k∣≥2)，预处理即可。
特别的，对于不足 nnn 位的数要特判。
