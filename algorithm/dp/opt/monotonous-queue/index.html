单调队列优化

2025-12-13

简介

单调队列通常用于维护连续区间最大最小值，可以在动态规划中用于维护状态间的转移。

单调队列优化多重背包

假设有 nnn 个物品，每个物品有 cic_ici​ 个，价值为 viv_ivi​，体积为 wiw_iwi​，背包容量为 mmm。
首先列出传统的 DP 转移方程：

f[i][j]=max⁡k>0{f[i−1][j−k⋅wi]+k⋅vi}f[i][j] = \max _{k>0} \left\{ f[i-1][j-k\cdot w_i]+k\cdot v_i \right\} f[i][j]=k>0max​{f[i−1][j−k⋅wi​]+k⋅vi​}

我们会发现，这个转移过程相当于每次向前跳若干的 wiw_iwi​，于是所有模 wiw_iwi​ 余数相同的 jjj 即在一个组中，将转移方程改写（这里的 jjj 有所改变）：

f[i][j⋅wi+d]=max⁡0≤k≤ci{f[i−1][j⋅wi+d−k⋅wi]+k⋅vi}f[i][j\cdot w_i+d] = \max_{0\leq k\leq c_i}\left\{ f[i-1][j\cdot w_i+d-k\cdot w_i] +k\cdot v_i\right\} f[i][j⋅wi​+d]=0≤k≤ci​max​{f[i−1][j⋅wi​+d−k⋅wi​]+k⋅vi​}

此时 max⁡\maxmax 内部 wiw_iwi​ 系数为 j−kj-kj−k，我们可以对照一下单调队列优化 DP 的标准形式：

f[i]=max⁡l≤k≤r{f[k]}+costf[i]=\max _{l\leq k\leq r} \left\{ f[k] \right\}+cost f[i]=l≤k≤rmax​{f[k]}+cost

由于 max⁡\maxmax 内部仅能与 kkk 有关，将其整个变形得到：

f[i][j⋅wi+d]=max⁡0≤k≤ci{f[i−1][(j−k)⋅wi+d]−(j−k)⋅vi}+j⋅vif[i][j\cdot w_i+d] = \max_{0\leq k\leq c_i} \left\{ f[i-1][(j-k)\cdot w_i+d]-(j-k)\cdot v_i \right\}+j\cdot v_i f[i][j⋅wi​+d]=0≤k≤ci​max​{f[i−1][(j−k)⋅wi​+d]−(j−k)⋅vi​}+j⋅vi​

这样，j−kj-kj−k 成为内部的 kkk，而其范围为 [j−ci,j][j-c_i,j][j−ci​,j]，满足单调队列优化形式。
可是还有一个问题，内部 j−kj-kj−k 的系数为 wiw_iwi​，也就导致不是区间内每个数都取，而是每 wiw_iwi​ 个取一次。
这样处理以后，我们可以复用一个单调队列，每更换一次余数清空一次即可。

AcWing 1091. 理想的正方形

AcWing 1091. 理想的正方形

我们考虑怎么求单独的一个正方形内的最大值。
可以将一个正方形拆成几行来看，只需要求这 nnn 行长度为 nnn 的最大值，每一行可以用单调队列求出当前这个元素左侧 nnn 个元素的最大值（包含自己）。
这个时候每一个正方形的最大值可以转换为最右边一列的最大值，所以我们还需要对最右边一列做一遍单调队列。
