最大公因数

2026-04-21

定义

公因数是指一个数是一组数中每个数的因数。
最大公因数是指公因数中最大的那个。

通常写作 gcd⁡(a,b)\gcd(a,b)gcd(a,b) 或 (a,b)(a,b)(a,b)。

欧几里得算法

对于两个数 aaa 和 bbb（a>ba>ba>b），怎么求它们的最大公因数呢？
显然，若 a mod b=0a\bmod b=0amodb=0，则 bbb 为两者最大公因数；若 a mod b≠0a\bmod b\neq 0amodb=0，则 gcd⁡(b,a mod b)\gcd(b,a\bmod b)gcd(b,amodb) 即为最大公因数，下面给出证明。

证明 gcd(a,b)=gcd(b,a mod b)gcd(a,b)=gcd(b,a\bmod b)gcd(a,b)=gcd(b,amodb)

令 a=k×b+c (0≤c<b)a=k\times b+c\ (0\leq c<b)a=k×b+c (0≤c<b)。

先证充分性。
设正整数 ddd 满足 d∣ad\mid ad∣a 且 d∣bd\mid bd∣b。
由于 c=a−k×bc=a-k\times bc=a−k×b，则有 cd=ad−k×bd\frac c d=\frac a d-k\times \frac b ddc​=da​−k×db​，此时右边都为整数，则 cd\frac c ddc​ 也为整数，即 d∣cd\mid cd∣c 成立。
因此，gcd⁡(a,b)⇒gcd⁡(b,a mod b)\gcd(a,b)\Rightarrow\gcd(b,a\bmod b)gcd(a,b)⇒gcd(b,amodb)。

再证必要性。
设正整数 ddd 满足 d∣cd\mid cd∣c 且 d∣bd\mid bd∣b。
由于 c+k×b=ac+k\times b=ac+k×b=a，则有 cd+k×bd=ad\frac c d+k\times \frac b d=\frac a ddc​+k×db​=da​，此时左边都为整数，则 ad\frac a dda​ 也为整数，即 d∣ad\mid ad∣a 成立。
因此，gcd⁡(b,a mod b)⇒gcd⁡(a,b)\gcd(b,a\bmod b)\Rightarrow\gcd(a,b)gcd(b,amodb)⇒gcd(a,b)。

那么复杂度怎么样呢？我们来讨论一下：

  1. 若 a<ba<ba<b，递归 gcd⁡(b,a)\gcd(b,a)gcd(b,a)。
  2. 若 a≥ba\geq ba≥b，递归 gcd⁡(b,a mod b)\gcd(b,a\bmod b)gcd(b,amodb)，显然 aaa 大小至少缩小一倍，而每发生一次 111，一定接着发生 222，即最多递归 log⁡V\log VlogV 层。

即复杂度为 O(log⁡min⁡(a,b))O(\log \min(a,b))O(logmin(a,b))。

Stein 算法

由于大整数的除法及取模操作及其昂贵，而减法与位移操作很快，所以 Stein 算法诞生了。

具体原理是根据奇偶性进行分类：

  1. 若 aaa 和 bbb 都是偶数，显然 gcd⁡(a,b)=2×gcd⁡(a2,b2)\gcd(a,b)=2\times\gcd(\frac a 2,\frac b 2)gcd(a,b)=2×gcd(2a​,2b​)。
  2. 若 aaa 为偶数，bbb 为奇数，显然 gcd⁡(a,b)=gcd⁡(a2,b)\gcd(a,b)=\gcd(\frac a 2,b)gcd(a,b)=gcd(2a​,b)。
  3. 若 aaa 和 bbb 都是奇数，显然 gcd⁡(a,b)=gcd⁡(∣a−b∣,min⁡(a,b))\gcd(a,b)=\gcd(\left|a-b\right|,\min(a,b))gcd(a,b)=gcd(∣a−b∣,min(a,b))。

时间复杂度为 O(log⁡V)O(\log V)O(logV)。

LL exgcd(LL a , LL b , LL &x , LL &y) {
    if (!b) {
        x = 1; y = 0;
        return a;
    }
    LL d = exgcd(b , a % b , y , x);
    y -= a/b*x;
    return d;
}
