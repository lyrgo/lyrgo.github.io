质数

2026-04-14

简介

质数是指大于 111 的自然数，且其仅有自己与 111 两个约数。

定理

1∼n1\sim n1∼n 内质数个数约为 nln⁡n\frac{n}{\ln n}lnnn​ 个。

根据唯一分解定理，N=p1k1p2k2⋯pnknN=p_1^{k_1}p_2^{k_2}\cdots p_n^{k_n}N=p1k1​​p2k2​​⋯pnkn​​。
约数个数：(k1+1)×(k2+1)×⋯×(kn+1)(k_1+1)\times(k_2+1)\times\cdots\times(k_n+1)(k1​+1)×(k2​+1)×⋯×(kn​+1)。
int 范围内，约数个数最多的数（191745944200191745944200191745944200 和 232113511400232113511400232113511400）有 153615361536 个约数。
约数和：(p10+p11+⋯+p1k1)×(p20+p21+⋯+p2k2)×⋯×(pn0+pn1+⋯+pnkn)(p_1^0+p_1^1+\cdots+p_1^{k_1})\times(p_2^0+p_2^1+\cdots+p_2^{k_2})\times\cdots\times(p_n^0+p_n^1+\cdots+p_n^{k_n})(p10​+p11​+⋯+p1k1​​)×(p20​+p21​+⋯+p2k2​​)×⋯×(pn0​+pn1​+⋯+pnkn​​)。

方法

试除法

判定一个数是否为质数：
显然， nnn 的因子是成对出现的，若一个为 ddd，则另一个为 nd\frac n ddn​，所以判定质数是只需要找 d≤ndd\leq \frac n dd≤dn​ 的数，即 d≤nd \leq \sqrt nd≤n​。
时间复杂度 O(n)O(\sqrt n)O(n​)。

找一个数的所有约数： 与判定相同，也是 d≤nd\leq \sqrt nd≤n​，每次对当前的 ddd 一直取出直到没有 ddd 这个约数，全部判断一遍即可。
但是，按照这种方法最坏复杂度为 O(n)O(n)O(n)，考虑将其优化掉，我们发现，在 nnn 中最多只有一个大于 n\sqrt nn​ 的质因子（有两个相乘一定大于 nnn），因此，我们可以只枚举到 n\sqrt nn​，最后剩下的数若不为 111 则为这个质因子。
时间复杂度 O(log⁡n)∼O(n)O(\log n)\sim O(\sqrt n)O(logn)∼O(n​)。

质数筛

判定从 1∼n1\sim n1∼n 的所有质数。

普通筛法：
对于每个 ddd，判定 ddd 的不大于 nnn 的倍数（不为 ddd 本身）并将其标记，容易发现，最后没被标记的数即为质数。
这个算法时间复杂度为 n2+n3+⋯+nn\frac n 2+\frac n 3+\cdots+\frac n n2n​+3n​+⋯+nn​，形如调和级数（lim⁡n→+∞n2+n3+⋯+nn=ln⁡n+c\lim\limits_{n\rightarrow +\infty} \frac n 2+\frac n 3+\cdots+\frac n n=\ln n+cn→+∞lim​2n​+3n​+⋯+nn​=lnn+c，ccc 为常数），即小于 O(nlog⁡n)O(n\log n)O(nlogn)。

埃拉托斯特尼筛法（埃氏筛）：
在此基础上，每个数我们都进行了判断倍数其实是多余的，可以只对不大于 n\sqrt nn​ 的质数进行判断，复杂度为 O(nlog⁡log⁡n)O(n\log \log n)O(nloglogn)。

线性筛（欧拉筛）：
线性筛主要思想是保证每个数只被其最小质因子标记。
具体的，若当前数 ddd 为质数表（单调递增）中某个数 ppp 的倍数，如果继续筛 ppp 的下一个数 p′p'p′，则由于 ddd 为 ppp 倍数，则在新数 d×p′d\times p'd×p′ 中一定包含 ppp 这个因数，又因为 p<p′p<p'p<p′，即 d×p′d\times p'd×p′ 的最小质因数实际为 ppp，所以应在 ddd 为 ppp 倍数是结束这一轮标记。
时间复杂度 O(n)O(n)O(n)。
