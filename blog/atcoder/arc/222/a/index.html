ARC222A Colorful Intervals

构造

2026-06-24

题目大意

给定一个 nnn，要求你构造一个正整数序列 AAA 满足 mmm 条限制，每条限制形式为 (li,ri)(l_i,r_i)(li​,ri​)，要求为 Ali∼AriA_{l_i}\sim A_{r_i}Ali​​∼Ari​​ 每个数字互不相同。

多组测试数据。
数据范围：1≤T≤105,  1≤n≤2×105,  1≤m≤2×105,  ∑n≤2×105,  ∑m≤2×1051\leq T\leq 10^5,\; 1\leq n\leq 2\times 10^5,\; 1\leq m\leq 2\times 10^5,\; \sum n\leq 2\times 10^5,\; \sum m\leq 2\times 10^51≤T≤105,1≤n≤2×105,1≤m≤2×105,∑n≤2×105,∑m≤2×105。

题解

已经没有人类了。
显然答案有一个下界 k=max⁡(ri−li+1)k=\max(r_i-l_i+1)k=max(ri​−li​+1)，考虑以下构造：1,2,3,⋯ ,k,1,2,3,⋯k,⋯1,2,3,\cdots,k,1,2,3,\cdots k,\cdots1,2,3,⋯,k,1,2,3,⋯k,⋯，显然满足题目要求。
所以 kkk 就是答案。

代码

#include <iostream>
#include <algorithm>
#include <cstdio>
#include <cstring>
#include <cmath>

using namespace std;

void solve() {
	int n , m;
	scanf("%d%d" , &n , &m);
	int ans = 0;
	while(m --) {
		int l , r;
		scanf("%d%d" , &l , &r);
		ans = max(ans , r-l+1);
	}
	for(int i = 1 ; i <= n ; i ++)
		printf("%d " , i%ans+1);
	puts("");
	return;
}

int main() {
	int T; scanf("%d" , &T);
	while(T --) solve();
	return 0;
}
