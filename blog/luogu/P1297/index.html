P1297 [国家集训队] 单选错位

概率与期望

2026-05-10

题目大意

给定 nnn 道题目，每个题目有 aia_iai​ 个选项，每个选项被选择的概率相同，但是上一个题的选项会被填写到下一个题，问期望正确的题的个数。
数据范围： 1≤n≤107,1≤ai≤1081\leq n\leq 10^7,1\leq a_i\leq 10^81≤n≤107,1≤ai​≤108。

题解

设随机变量 Xi∈{0,1}X_i\in \{0,1\}Xi​∈{0,1} 代表第 iii 道题是否正确，则显然正确题目总数 X=∑i=1nXiX=\sum _{i=1}^n X_iX=∑i=1n​Xi​，利用期望线性性，则有 E(X)=∑i=1nE(Xi)E(X)=\sum _{i=1}^n E(X_i)E(X)=∑i=1n​E(Xi​)，容易发现，对于每道题做对的期望实际就是做对这道题的概率。
于是，第 iii 题正确答案在 aia_iai​ 中取，而填的答案在 ai−1a_{i-1}ai−1​ 中取，则有：

P(Xi=1)=1ai×1ai−1×min⁡(ai,ai−1)=1max⁡(ai,ai−1)P(X_i=1)=\frac 1 {a_i}\times \frac 1 {a_{i-1}}\times \min(a_i,a_{i-1})=\frac 1 {\max(a_i,a_{i-1})} P(Xi​=1)=ai​1​×ai−1​1​×min(ai​,ai−1​)=max(ai​,ai−1​)1​

代码

#include <iostream>
#include <algorithm>
#include <cstdio>
#include <cstring>
#include <cmath>

using namespace std;

typedef long long LL;
typedef long double LD;

const int N = 1e7+10;

int n;
LL a[N];

int main() {
	int A , B , C;
	scanf("%d%d%d%d%lld" , &n , &A , &B , &C , &a[1]);
	for(int i = 2 ; i <= n ; i ++)
		a[i] = (a[i-1]*A+B)%100000001ll;
	for(int i = 1 ; i <= n ; i ++)
		a[i] = a[i] % C + 1;
	
	LD res = 1.0l / max(a[1] , a[n]);
	for(int i = 2 ; i <= n ; i ++) {
		res += 1.0l / max(a[i] , a[i-1]);
	}
	printf("%.3Lf\n" , res);
	return 0;
}
