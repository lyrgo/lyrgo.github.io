Luogu P14507 缺零分治 mexdnc

题目大意

题目传送门

给定正整数 nnn 和 nnn 个二元组 (bi,ai)(b_i,a_i)(bi​,ai​)，表示有 bib_ibi​ 个大小为 aia_iai​ 的数。
定义 mex\text{mex}mex 表示一个集合中最小未出现的自然数。
你需要将这 ∑i=1nbi\sum\limits_{i=1}^{n}b_ii=1∑n​bi​ 个数分为 k(k≥1)k(k\geq 1)k(k≥1) 个可重集合，使得这 kkk 个集合的 mex\text{mex}mex 和恰好为 mmm，求 kkk 的最小值。
给定 TTT 组询问，每次给定一个 mmm，求最小的 kkk，若无解则输出 −1-1−1。

Solution

很容易注意到有些 aia_iai​ 是没有用的，我们考虑如何将其去除。
从题目中可以发现，aia_iai​ 是升序给我们的，那么有用的 aia_iai​ 一定是形如 0,1,...,x{0,1,...,x}0,1,...,x 的一个排列。

结论

找到最小的 iii 使得 ai+1≠ai+1a_i+1\neq a_{i+1}ai​+1=ai+1​（若不存在则 iii 为 nnn），则 ai+1a_{i+1}ai+1​ 到 ana_nan​ 都没有用

我们令 nnn 为上面的 iii。 于是问题变成了对于 i∈[1,n]i\in [1,n]i∈[1,n]，有 bib_ibi​ 个 i−1i-1i−1。

先来考虑无解的情况，分讨一下有两种：

  1. m=0m=0m=0，若 n=0n=0n=0，那么一定有一个集合被分到 000，即 mex≥1\text{mex}\geq 1mex≥1，总和一定不是 000，无解。
  2. 令目前所有数能凑出来的 mex\text{mex}mex 的和的最大值为 xxx，若 m>xm>xm>x，则无解。

考虑怎么求最大和：

  • 一个集合中的所有数当且仅当出现一次，否则多余的数不产生贡献。
  • 如果可以组成一个 [0,u−1][0,u-1][0,u−1] 的序列，那么将其拆分为 [0,v−1](v<u)[0,v-1](v < u)[0,v−1](v<u) 一定不优，因为 [v,u−1][v,u-1][v,u−1] 被浪费了。

所以我们只需要每次找到最大的 uuu 满足可以构成 [0,u−1][0,u-1][0,u−1] 的一个序列，产生 uuu 的贡献，重复这个步骤就能得到最大值。

考虑怎么进行代码实现，我们令 fi=min⁡j=1ibjf_i=\min\limits_{j=1}^{i}b_jfi​=j=1mini​bj​，gig_igi​ 为最小的满足 bj=fib_j=f_ibj​=fi​ 的 jjj。 进行如下步骤：

  1. 初始化
    初始时令 p=np=np=n，d=0d=0d=0（ppp 为上面最大的 uuu，ddd 为删掉的 000 的个数）
    由于删掉的 000 的数其实就是序列总数，所以 ddd 就是删除的序列数。
  2. 计算贡献
    令 c=fp−dc=f_p-dc=fp​−d 即为能够取出多少个 [0,p−1][0,p-1][0,p−1] 的序列，产生的贡献就是 p×cp\times cp×c。
  3. 更新变量
    拿出 ccc 个序列后，d←d+cd\leftarrow d+cd←d+c。
    这个时候的 gp−1g_p-1gp​−1 已经用完（gpg_pgp​ 在原数组对应 gp−1g_p-1gp​−1 这个数），所以 gp−1g_p-1gp​−1 到 p−1p-1p−1 之间剩的数都没有用了，令 p←gp−1p\leftarrow g_p-1p←gp​−1（注意这个时候指的是下标）
  4. 完成
    重复这个步骤，直到 p=0p=0p=0。

这样我们得到了最大值。

如何计算 fff 和 ggg？

很明显要考虑递推。
如果当前位置 iii 的 bi<fi−1b_i<f_{i-1}bi​<fi−1​，那么 fi=bif_i=b_ifi​=bi​，gi=ig_i=igi​=i。
如果当前位置 iii 的 bi≥fi−1b_i\geq f_{i-1}bi​≥fi−1​，那么 fi=fi−1f_i=f_{i-1}fi​=fi−1​，gi=gi−1g_i=g_{i-1}gi​=gi−1​

这样，我们成功处理了无解的情况。

当前情况一定有解。
我们把所有构成最大值的数组存下来，记作 eee，则 eie_iei​ 代表一个 [0,ei−1][0,e_i-1][0,ei​−1] 的序列。
同时，为了用尽量少的集合，我们应该尽可能选择 mex\text{mex}mex 大的集合。
怎么选呢？事实上，eie_iei​ 为单调不增的。
这样就好办了，只需要每次找到一个 iii 满足 ∑j=1iei≤m\sum\limits_{j=1}^i e_i\leq mj=1∑i​ei​≤m 且 ∑j=1i+1ei>m\sum\limits_{j=1}^{i+1} e_i>mj=1∑i+1​ei​>m，则为了凑 mmm，可以将第 i+1i+1i+1 个集合拆开，只留 [0,m−∑j=1iei][0,m-\sum\limits_{j=1}^i e_i][0,m−j=1∑i​ei​]（若 m=∑j=1ieim=\sum\limits_{j=1}^i e_im=j=1∑i​ei​，可以直接不要这个集合），那么前 i+1i+1i+1 个集合的 mex\text{mex}mex 和刚好为 mmm。
至于多出来的东西，可以丢到前面的集合中让其重复，这样就不会对答案产生贡献了。（其实可以全都丢到 e1e_1e1​，因为 eie_iei​ 单调不增）

这个时候的答案就是 i+[m=∑j=1iei]i+[m=\sum\limits_{j=1}^i e_i]i+[m=j=1∑i​ei​]。
时间复杂度 O(Tmax⁡bi)O(T\max b_i)O(Tmaxbi​)，空间复杂度 O(max⁡bi)O(\max b_i)O(maxbi​)。

很显然还没完事。。。
上面做法的问题在于，最大化时一定会带来一堆重复的 eie_iei​，不如开个 fff，让 fif_ifi​ 表示 eie_iei​ 有多少个，空间复杂度 O(n)O(n)O(n)。

Code

#include <cstdio>
#include <algorithm>
#include <iostream>
#include <cmath>
#include <cstring>

using namespace std;

typedef long long LL;

struct Node {LL a , b; };

const int N = 1e5+10;

int n , q , cnt;
LL A[N] , B[N];
LL f[N] , g[N];
LL w[N] , sw[N] , sc[N];

void slove() {
	memset(w , 0 , sizeof w);
	memset(sw , 0 , sizeof sw);
	memset(sc , 0 , sizeof sc);
	memset(f , 0 , sizeof f);
	memset(g , 0 , sizeof g);
	scanf("%d%d" , &n , &q); cnt = 0;
	int tmp = n;
	for(int i = 1 ; i <= n ; i ++) {
		scanf("%lld%lld" , &A[i] , &B[i]);
		if(tmp == n && A[i] != i-1) {
			tmp = i-1;
		}
	}
	n = tmp;

    if(n == 0) {
        while(q --) {
            LL m; scanf("%lld" , &m);
            if(m) puts("-1");
            else puts("1");
        }
        return;
    }
	
	f[1] = B[1]; g[1] = 1;
	for(int i = 2 ; i <= n ; i ++) {
		f[i] = f[i-1]; g[i] = g[i-1];
		if(B[i] < f[i]) {
			f[i] = B[i];
			g[i] = i;
		}
	}
	
	int p = n; LL d = 0;
	while(p) {
		LL c = f[p]-d;
		
		++ cnt;
		w[cnt] = p;
		sc[cnt] = sc[cnt-1] + c;
		sw[cnt] = sw[cnt-1] + c*p;
		
		p = g[p]-1; d += c;
	}
	
	while(q --) {
		LL m; scanf("%lld" , &m);
		if((m == 0 && n != 0) || m > sw[cnt]) {
			puts("-1");
			continue;
		}
		if(m < w[1]) {
			puts("2");
			continue;
		}
		
		int l = 0 , r = cnt , mid , pos = 0;
		while(l <= r) {
			mid = (l+r) >> 1;
			if(sw[mid] <= m) {
				pos = mid;
				l = mid+1;
			} else {
				r = mid-1;
			}
		}
		
		LL res = 0;
		if(pos == 0) {
            res = (m+w[1]-1)/w[1];
		} else if(pos == cnt || sw[pos] == m) {
			res = sc[pos];
		} else if(w[pos+1] == 0) {
			res = sc[pos];
		} else {
			res = (m-sw[pos]+w[pos+1]-1)/w[pos+1]+sc[pos];
		}
		printf("%lld\n" , res);
	}
}

int main() {
	int T; scanf("%d" , &T);
	while(T --) slove();
	return 0;
}
