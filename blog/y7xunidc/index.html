CF1601D Difficult Mountain

2026-06-26

题目大意

有 nnn 个人要登山，初始攀登难度为 ddd。
每个人有两个属性：技巧 sss 和整洁度 aaa。
技巧为 sss 的人能登上当前难度 ppp 的山当且仅当 p≤sp \le sp≤s。
当一位整洁度为 aaa 的人成功登山后，山的难度会变为 max⁡(p,a)\max(p, a)max(p,a)。

你可以任意安排这些人的登山顺序，每个人若轮到且能登山则一定会登，求最多能让多少人成功登山。

数据范围：1≤n≤5×105,  0≤d≤109,  0≤si,ai≤1091 \le n \le 5 \times 10^5,\; 0 \le d \le 10^9,\; 0 \le s_i, a_i \le 10^91≤n≤5×105,0≤d≤109,0≤si​,ai​≤109。

题解

首先我们排除所有 si<ds_i<dsi​<d 的人，显然他们没有用。
考虑贪心策略：将所有人以 max⁡(ai,si)\max(a_i,s_i)max(ai​,si​) 排序，如果相同则以 sis_isi​ 升序排序。

证明：

  1. 若 max⁡(ai,si)=si\max(a_i,s_i)=s_imax(ai​,si​)=si​，即 ai≤sia_i\leq s_iai​≤si​ 时：
    考虑在 iii 以前的人，要么 max⁡(sj,aj)<si\max(s_j,a_j)<s_imax(sj​,aj​)<si​，显然 d<sid<s_id<si​；要么 max⁡(sj,aj)=si\max(s_j,a_j)=s_imax(sj​,aj​)=si​，则 d≤sid\leq s_id≤si​。
    所以，有 si≥ds_i\geq dsi​≥d，此时可以直接更新答案。

    直接更新为什么更优？显然我们要考虑 iii 以后的人，有两种情况：

    • iii 以后的全部取 sjs_jsj​，此时由于 sj≥si≥ais_j\geq s_i\geq a_isj​≥si​≥ai​，不会产生影响。
    • iii 以后取一部分 sjs_jsj​，然后遇到一个取 aka_kak​ 的人，记为 kkk。
      如果 sk≥ais_k\geq a_isk​≥ai​，显然 kkk 也可以去，不影响。
      如果 sk<ais_k<a_isk​<ai​，要让 kkk 去必须至少让 iii 不去，答案不会更大，又因为 ak≥aia_k\geq a_iak​≥ai​，ddd 不会变小，一定不会更优。

  2. 若 max⁡(ai,si)=ai\max(a_i,s_i)=a_imax(ai​,si​)=ai​，即 ai≥sia_i\geq s_iai​≥si​ 时：
    与前面一样，有 d≤aid\leq a_id≤ai​。
    此时若 si≥ds_i\geq dsi​≥d，可以直接去。
    若 si<ds_i<dsi​<d，显然不能去，如果一定要去的话至少要让前面的一个人不去，ddd 不会更小，答案也不会更大，一定不会更优。
