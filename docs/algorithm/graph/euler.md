## 基础概念
**欧拉路径**：从某个点出发，经过所有边一次。  
**欧拉回路**：从某个点出发，经过所有边一次，回到这个点。  

对于一个**联通的**无向图的欧拉路径，若起点与终点不同，则它们都应该是奇数度的，其他点必须是偶数度；如果起点终点相同，则所有点都是偶数点。  
因此，一个联通的无向图中存在欧拉路径的充要条件是仅有 $0$ 或 $2$ 个奇数点。  
同理，一个联通的无向图中存在欧拉回路的充要条件是没有奇数点。  

::: details 证明
考虑构造一个 DFS。  
由于除了起点终点以外其余点一定是偶数点，那么从起点出发，每个点一定在进去以后可以出来，具体路径类似于一条边后有几个相邻的环（共用某几个点，且不一定绕圈，可以来回走），在进入终点后出不来（起点终点重合则此时出去后一定要回来）。  
换句话说，我们可以先找到从起点到终点的最短路后每次加环，便可以构造出合法方案。  
:::

对于一个**联通的**有向图的欧拉路径，若起点与终点不同，则起点出度比入度大 $1$，终点入度比出度大 $1$，其余点出度等于入度；若起点与终点相同，则所有点的入度应等于出度。  
同理，一个联通的有向图中存在欧拉回路的充要条件是所有点出度等于入度。  

### 细节
上面所述的 DFS 做法最优显然是 $O(n+m)$ 的，但如果一个点有多个自环，最差会退化成 $O(m^2)$，需要进行一些优化。  
具体的，我们可以在走过一条边的时候不只是把它标记，而是直接把它删掉（无向图记得把它的反向边一起删了）。

## 例题
### Acwing 1123. 铲雪车{#acwing-1123}
<LinkCard title="Acwing 1123. 铲雪车" href="https://www.acwing.com/problem/content/1125/">

读完题，我们会发现任意一段路径的两端的点入度和出度都会加 $1$，也就是说整个图的点的度数都是偶数，由于扫雪车最开始一定位于道路上，所以我们一定可以找到一个欧拉回路。  
此时，答案其实是 $\frac{sum}{20\text{km/h}}$（其中 $sum$ 是路径长度和），压根不是图论问题。
