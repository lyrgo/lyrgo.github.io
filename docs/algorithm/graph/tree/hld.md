## 简介
令一棵树的根节点为 $u$，对于它的所有子节点记作 $v_i$，可以取一个子树大小最大的 $v_{max}$，将其称作**重边**，其他的边称作轻边，容易发现，很多重边会连在一起，成为重链。可以证明，一棵树中，最多有 $\log n$ 条重链（可以理解为每走一条重边就会减少至少 $\frac{x}{2}$ 个节点）。  
由于重边个数是 $\log$ 级别的，所以重链也是 $\log$ 级别的。
