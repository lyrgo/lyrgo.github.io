## 定义
公平组合游戏需要严格满足以下条件：
1.  两名玩家轮流行动。
2.  局面对双方公开且无隐藏信息，且没有等概率随机元素。  
3.  同一个状态下两名玩家能做出的操作集合是相同的。  
4.  游戏必然在有限步内结束，通常约定无法操作的玩家为负。  

## 图论建模
*   节点：当前局面
*   连边：可行操作

如果这么建模，当一个玩家面对出度为 $0$ 的点，这个玩家就输了。  
在 DAG 建模中，所有节点被严格归为两类：  
1.  P 态：下一个操作的玩家必输。  
2.  N 态：下一个操作的玩家必胜。  

相应的，还有它们的转移：
1.  若一个节点出度为 $0$，则该节点为 P 态。  
2.  若一个节点存在一个为 P 态的后继状态，则该节点为 N 态。  
3.  若一个节点所有后继状态都为 N 态，则该节点为 P 态。  

## Nim 游戏
桌面上有 $n$ 堆石子，第 $i$ 堆有 $a_i$ 个石子，两名玩家轮流操作，每次可以从一堆中取走任意多的石子，谁取完最后一个谁胜利，给定每堆石子数量，怎么判断先手必胜还是先手必输？  

### Bouton 定理
定义当前局面的 Nim 和（异或和）为 $S=a_1\oplus a_2\oplus\cdots\oplus a_n$。  
*   若 $S=0$，则当前局面为 P 态，即先手必败。  
*   若 $S\neq 0$，则当前局面为 N 态，即先手必胜。  

接下来是 Bouton 定理的证明。  
1.  对于全空局面，显然为 P 态。  
2.  P 态的转移：  
    假设当前局面 $S=0$，显然玩家如何操作后产生的新异或和 $S'\neq 0$。  
3.  N 态的转移：  
    显然可以将提供 $S$ 最高位 $1$ 的那一堆取走最高位 $1$ 与剩下 $1$ 的差，此时 $S=0$。
