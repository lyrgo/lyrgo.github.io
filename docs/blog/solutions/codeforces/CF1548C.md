## 题目大意
<LinkCard title="CF1548C The Three Little Pigs" href="https://codeforces.com/problemset/problem/1548/C"/>
给定 $n$ 和若干 $x$，对每个 $x$ 求 $\sum^n_{i=1}\binom{3i}{x}$。  
其中 $1\leq n\leq 10^{6}$，$1\leq x\leq 3n$。  

## 题解
这里需要用到一个小技巧，对于在求和中的组合数，可以利用递推公式将其拆开，然后化简出求和的递推公式。  

显然我们需要拆式子：
$$
\begin{aligned}
\binom{3i}{x}
& = \binom{3i-1}{x}+\binom{3i-1}{x-1} \\
& = \binom{3i-2}{x}+\binom{3i-2}{x-1}+\binom{3i-2}{x-1}+\binom{3i-2}{x-2} \\
& = \binom{3i-3}{x}+3\times \binom{3i-3}{x-1}+3\times \binom{3i-3}{x-2}+\binom{3i-3}{x-3}  
\end{aligned}
$$
设 $S(x)=\sum\limits^n_{i=1}\binom{3i}{x}$，接下来找一下 $\sum\limits_{i=1}^{n}\binom{3i-3}{y}$ 与 $S(3x)$ 的关系：
$$
\sum_{i=1}^{n}\binom{3i-3}{y}=\binom{0}{y}+\binom{3}{y}+\cdots+\binom{3n-3}{y}=S(y)-\binom{3n}{y}+\binom{0}{y}
$$
并且有两个恒等式：
$$
\begin{aligned}
& \binom{3n}{x}+3\times \binom{3n}{x-1}+3\times \binom{3n}{x-2} +\binom{3n}{x-3}=\sum_{k=0}^{3}\binom{3}{k}{\times \binom{3n}{x-k}}=\binom{3n+3}{x}  \\
& \binom{0}{x} +\binom{0}{x-1} +\binom{0}{x-2} +\binom{0}{x-3}=\sum_{k=0}^{3}\binom{3}{k}\times \binom{3-3}{x-k} = \binom{3}{x} 
\end{aligned}
$$
将其带入，消去 $S(x)$，得到：
$$
\binom{3n+3}{x} -\binom{3}{x+1} =3\times S(x-1)+3\times S(x-2)+S(x-3)
$$
将 $x$ 替换为 $x+1$，变形得到：
$$
S(x)=\frac{1}{3} \left[ \binom{3n+3}{x+1} -\binom{3}{x+1} -S(x-2) \right]-S(x-1)
$$
这便是能够直接转移的公式，只需要线性处理逆元即可。  
对于边界，显然有 $S(0)=n$，$S(1)=\frac{3n(n+1)}{2}$。  

## 代码
``` cpp :collapsed-lines
#include <iostream>
#include <algorithm>
#include <cstdio>
#include <cstring>
#include <cmath>

using namespace std;

typedef long long LL;

const int N = 1e6+10;
const LL M = 1e9+7;

int n , m;
LL fac[N*3] , inv[N*3];
LL res[N*3];

LL qmi(LL a , LL b) {
	LL res = 1;
	while(b) {
		if(b & 1) res = res * a % M;
		b >>= 1; a = a * a % M;
	}
	return res;
}

int main() {
	scanf("%d%d" , &n , &m);
	
	fac[0] = inv[0] = 1;
	fac[1] = inv[1] = 1;
	for(int i = 2 ; i <= n*3+3 ; i ++) {
		fac[i] = fac[i-1]*i % M;
	}
	inv[n*3+3] = qmi(fac[n*3+3] , M-2);
	for(int i = n*3+2 ; i >= 1 ; i --) {
		inv[i] = inv[i+1]*(i+1)%M;
	} 
	
	res[0] = n; res[1] = (3ll*n*(n+1) % M)*inv[2] % M;
	for(int i = 2 ; i <= n*3 ; i ++) {
		LL t1 = 0;
		if(i <= 2) t1 = (fac[3]*inv[2-i]%M)*inv[i+1]%M;
		LL t2 = (fac[n*3+3]*inv[i+1] % M)*inv[n*3-i+2] % M;
		res[i] = (inv[3]*2ll%M)*((t2-t1-res[i-2]+M)%M)%M;
		res[i] = (res[i]-res[i-1]+M)%M;
	}
	
	while(m --) {
		int x; scanf("%d" , &x);
		printf("%lld\n" , res[x]);
	}
	return 0;
}
```
