## 题目大意
有 $n$ 个人要登山，初始攀登难度为 $d$。  
每个人有两个属性：技巧 $s$ 和整洁度 $a$。  
技巧为 $s$ 的人能登上当前难度 $p$ 的山当且仅当 $p \le s$。  
当一位整洁度为 $a$ 的人成功登山后，山的难度会变为 $\max(p, a)$。  

你可以任意安排这些人的登山顺序，每个人若轮到且能登山则一定会登，求最多能让多少人成功登山。

数据范围：$1 \le n \le 5 \times 10^5,\; 0 \le d \le 10^9,\; 0 \le s_i, a_i \le 10^9$。

## 题解
首先我们排除所有 $s_i<d$ 的人，显然他们没有用。  
考虑贪心策略：将所有人以 $\max(a_i,s_i)$ 排序，如果相同则以 $s_i$ 升序排序。

证明：  
1.  若 $\max(a_i,s_i)=s_i$，即 $a_i\leq s_i$ 时：  
    考虑在 $i$ 以前的人，要么 $\max(s_j,a_j)<s_i$，显然 $d<s_i$；要么 $\max(s_j,a_j)=s_i$，则 $d\leq s_i$。  
    所以，有 $s_i\geq d$，此时可以直接更新答案。  

    直接更新为什么更优？显然我们要考虑 $i$ 以后的人，有两种情况：  
    *   $i$ 以后的全部取 $s_j$，此时由于 $s_j\geq s_i\geq a_i$，不会产生影响。  
    *   $i$ 以后取一部分 $s_j$，然后遇到一个取 $a_k$ 的人，记为 $k$。  
        如果 $s_k\geq a_i$，显然 $k$ 也可以去，不影响。  
        如果 $s_k<a_i$，要让 $k$ 去必须至少让 $i$ 不去，答案不会更大，又因为 $a_k\geq a_i$，$d$ 不会变小，一定不会更优。  
2.  若 $\max(a_i,s_i)=a_i$，即 $a_i\geq s_i$ 时：  
    与前面一样，有 $d\leq a_i$。  
    此时若 $s_i\geq d$，可以直接去。  
    若 $s_i<d$，显然不能去，如果一定要去的话至少要让前面的一个人不去，$d$ 不会更小，答案也不会更大，一定不会更优。
