## 题目大意
有一个长度为 $n$ 且每一位都是 $0$ 的数，接下来会在 $[0,2^n)$ 之间等概率给出整数 $x$ 并与原数按位或，则这个数变为 $2^n-1$ 的抽取数的数量的期望值为多少？  
数据范围：$1\leq n\leq 20$。  

## 前置知识
[Max-min 容斥](/algorithm/math/combinatorics/inclusion-exclusion-principle/#max-min-容斥){.readmore}
[子集和 DP](/algorithm/basic/prefix-sum/#子集和-dp){.readmore}

## 题解
首先考虑进行拆位，设 $S$ 为这 $n$ 个二进制位组成的集合，原数变成 $2^n-1$ 等价于所有位都变成 $1$，由于 $\max$ 不好求，考虑 Max-min 容斥，则有：
$$
\text{E}(\max(S))=\sum_{\substack{T\subseteq S\\T\neq \varnothing}}(-1)^{|T|-1}\text{E}(\min(T))
$$
其中 $\text{E}(\min(T))$ 表示 $T$ 中若干位变为 $1$ 的期望值。  
设抽到一个与 $T$ 有交集的数字概率为 $\text{P}(T)$，则 $\text E(\min(T))=\frac 1 {\text P(T)}$，直接求交集很难，正难则反，$\text P(T)=1-\text P(\text{抽到的数与 }T\text{ 交集为空集})$。  
也就是说，我们需要快速计算 $S/T$（$S$ 中除去 $T$）的子集，令其概率为 $Sum(S/T)$，考虑如何计算。  
由于直接处理复杂度为 $O(3^n)$，考虑使用高维前缀和，在 $O(n2^n)$ 复杂度内预处理出 $Sum$ 数组，最后带入容斥公式即可。  

## 代码
```cpp :collapsed-lines
#include <iostream>
#include <algorithm>
#include <cstdio>
#include <cstring>
#include <cmath>

using namespace std;

typedef double DB;

const int N = 20;

int n;
DB sum[1<<N];
int cnt[1<<N];

int main() {
	scanf("%d" , &n);
	for(int i = 0 ; i < (1<<n) ; i ++) {
		scanf("%lf" , &sum[i]);
		cnt[i] = cnt[i>>1]+(i&1);
	}
	for(int i = 0 ; i < n ; i ++)
		for(int j = 0 ; j < (1<<n) ; j ++)
			if((j >> i) & 1)
				sum[j] += sum[j^(1<<i)];
	
	DB res = 0;
	for(int i = 1 ; i < (1<<n) ; i ++) {
		DB x = 1.0-sum[((1<<n)-1)^i];
		if(x > 1e-8) {
			if(cnt[i] & 1) res += 1/x;
			else res -= 1/x;
		}
	}
	if(res < 1e-10) puts("INF");
	else printf("%.9lf\n" , res);
	return 0; 
}
```
