## 氯元素简介
氯元素在自然界中以单质形式存在，由于其最外层有 $7$ 个电子，它的化合价区间为 $[-1,+7]$，稳定化合价为 $-1$ 价。  

## 氯气
$\ce{Cl2}$ 为黄绿色，有刺激性气味的有毒气体，易液化（储存在钢瓶中），可溶于水，密度比空气大。  
$\ce{Cl2}$ 易得电子，具有强氧化性。

### 氯气与金属单质反应
由于 $\ce{Cl2}$ 具有强氧化性，所以不管是少量还是大量都会生成高价金属氯化物。  
*   $\ce{Na}$ 在 $\ce{Cl2}$ 中剧烈燃烧，释放大量的热，生成白烟：$\ce{2 Na + Cl_2 \xlongequal{\Delta}{} 2 NaCl}$  
*   $\ce{Fe}$ 在 $\ce{Cl2}$ 中剧烈燃烧，释放大量的热，生成**棕褐色**的烟：$\ce{2 Fe + 3 Cl2 \xlongequal{\Delta}{} 2 FeCl3}$  
*   $\ce{Cu}$ 在 $\ce{Cl2}$ 中剧烈燃烧，释放大量的热，生成**棕黄色**的烟：$\ce{Cu + Cl2 \xlongequal{\Delta}{} CuCl2}$  

### 氯气与非金属单质反应
*   $\ce{H2}$ 在 $\ce{Cl2}$ 中安静的燃烧，发出苍白色火焰，集气瓶口有白雾：$\ce{H2 + Cl2 \xlongequal{\Delta}{} 2 HCl}$  
*   $\ce{P}$ 在 $\ce{Cl2}$ 中燃烧，释放大量的热，生成白色的**烟和雾**：$\ce{2 P + 3 Cl2 \xlongequal{\Delta}{} 2 PCl3}$、$\ce{PCl3 + Cl2 <=> PCl5}$（前一个为液体，后一个为固体）。  

### 氯气与水反应
原理：水中所有元素均为稳价，$\ce{Cl2}$ 与 $\ce{H2O}$ 发生氧化还原反应时，$\ce{Cl2}$ 发生歧化反应，且为可逆反应。  
化学方程式：$\ce{Cl2 + H2O <=> HCl + HClO}$  
离子方程式：$\ce{Cl2 + H2O <=> H+ + Cl- + HClO}$  

新制氯水与液氯的组成比较：
|新制氯水（混合物）|液氯（纯净物）|
|-----|-----|
|“三分四离”：$\ce{H2O,Cl2,HClO,H+,ClO-,OH-}$|$\ce{Cl2}$|

也就是说，新制氯水中 $\ce{Cl2}$ 体现强氧化性，$\ce{HClO}$ 体现漂白性。  
验证方式：干燥的 $\ce{Cl2}$ 分别接触干燥的和湿润的有色布条。  

新制氯水与久置氯水的组成比较：  
新制氯水中的 $\ce{HClO}$ 不稳定：$\ce{2 HClO \xlongequal{光照}{} 2 HCl + O2 ^}$  
久置氯水：$\ce{H2O,H+,Cl-,OH-}$（跟稀盐酸差不多）。  

### 氯气与碱反应
$\ce{Cl2}$ 和**冷的** $\ce{NaOH}$ 溶液反应生成漂白液（有效成分为 $\ce{NaClO}$）。  
化学方程式：$\ce{Cl2 + 2 NaOH \xlongequal{}{} NaCl + NaClO + H2O}$（这个也是歧化反应）  
离子方程式：$\ce{Cl2 + 2 OH- \xlongequal{}{} Cl- + ClO- + H2O}$  

没错，这玩意会变质：$\ce{NaClO + CO2 + H2O \xlongequal{}{} NaHCO3 + HClO}$（遵循弱酸 $\ce{H2CO3}$ 制更弱酸 $\ce{HClO}$）。  
虽然但是，生成的 $\ce{HClO}$ 照样分解（）  

注意，加热时 $\ce{Cl}$ 化合价变化可能不遵循邻价规律，如 $\ce{3 Cl2 + 6 NaOH \xlongequal{\Delta}{} 5 NaCl + NaClO3 + 3 H2O}$  

### 氯气与其他化合物反应
:::table title="" align="center" copy=false max-content 
| 反应对象 | 化学方程式 | 离子方程式 |
| --- | --- | --- |
| 与 $\ce{H2S}$ 溶液反应 | $\ce{Cl2 + H2S \xlongequal{}{} S v + 2HCl}$ | $\ce{Cl2 + H2S \xlongequal{}{} S v + 2H+ + 2Cl-}$ |
| 与 $\ce{SO2}$ 水溶液反应 | $\ce{Cl2 + SO2 + 2H2O \xlongequal{}{} 2HCl + H2SO4}$ | $\ce{Cl2 + SO2 + 2H2O \xlongequal{}{} 4H+ + 2Cl- + SO4^2-}$ |
| 与 $\ce{KI}$ 溶液反应 | $\ce{Cl2 + 2KI \xlongequal{}{} I2 + 2KCl}$ | $\ce{Cl2 + 2I- \xlongequal{}{} I2 + 2Cl-}$ |
| 与 $\ce{KBr}$ 溶液反应 | $\ce{Cl2 + 2KBr \xlongequal{}{} Br2 + 2KCl}$ | $\ce{Cl2 + 2Br- \xlongequal{}{} Br2 + 2Cl-}$ |
| 与 $\ce{FeCl2}$ 溶液反应 | $\ce{Cl2 + 2FeCl2 \xlongequal{}{} 2FeCl3}$ | $\ce{Cl2 + 2Fe^{2+} \xlongequal{}{} 2Fe^{3+} + 2Cl^-}$ |
| 与 $\ce{NH3}$ 反应 | $\ce{8NH3 + 3Cl2 \xlongequal{}{} N2 + 6NH4Cl}$ {colspan=2}|
:::
其中与 $\ce{NH3}$ 的反应可以检测运输液氯的管道是否漏气（$\ce{NH4Cl}$ 为白色固体小颗粒）。
